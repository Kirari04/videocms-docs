# VideoCMS Documentation

This repository contains the documentation for [VideoCMS](https://github.com/Kirari04/videocms).

The documentation is built with [VitePress](https://vitepress.dev/).

## Development

To start the documentation locally for development:

```bash
bun install
bun run dev
```

The documentation will be available at `http://localhost:5173`.

## Build

To build the static site:

```bash
bun run docs:build
```

The output will be in `docs/.vitepress/dist`.
